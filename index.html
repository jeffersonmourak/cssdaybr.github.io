A Web tem evoluído de uma forma incrível nos últimos anos e notamos uma ênfase enorme nas funcionalidades do JavaScript e nas famosas APIs do HTML5, porém muitos esquecem que a Interface para se tornar elegante e utilizável precisa ser feita com um cuidado especial usando CSS.

Com o intuito de disseminar conhecimento relacionado às folhas de estilo e com o apoio da CSS Conf Brazil trazemos o CSSday, um dia imersivo para que possamos reconhecer a importância do CSS no cenário do Desenvolvimento Web.
